## CH3 k近邻法
[kNN](CH3/README.md)
- kNN是一种基本的分类与回归方法
- k值的选择, 距离度量及分类决策规则是kNN的三个基本要素.
## CH4 朴素贝叶斯法
[NB](CH4/README.md)
- 朴素贝叶斯法是基于贝叶斯定理与特征条件独立假设的分类方法.
1. IID->输入输出的联合概率分布
1. Bayes->后验概率最大的输出
- 平滑处理方案, lambda = 1 对应拉普拉斯平滑
